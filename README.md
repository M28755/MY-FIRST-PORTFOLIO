## Features

-  **Animated Hero Section** — Typing animation cycling through my roles (Web Developer, Data Scientist, Ethical Hacker, Cyber-Security Engineer)
-  **About Me Section** — Personal bio with animated skill progress bars
-  **Projects Showcase** — Cards displaying real GitHub projects with tags, descriptions, and links
-  **Contact Section** — Glassmorphism-styled contact form with a live code block displaying my contact info
-  **Smooth Animations** — Powered by Framer Motion with scroll-triggered `whileInView` animations throughout
-  **Responsive Design** — Fully responsive across all screen sizes
- **Social Links** — Direct links to Facebook, TikTok, Instagram, GitHub, and LinkedIn

## Getting Started

### Prerequisites
- [Node.js](https://nodejs.org/) v18 or higher
- npm v9 or higher

### Installation

```bash
# 1. Clone the repository
git clone https://github.com/M28755/MY-FIRST-PORTFOLIO.git

# 2. Navigate into the project directory
cd MY-FIRST-PORTFOLIO

# 3. Install dependencies
npm install

# 4. Start the development server
npm run dev
```

The app will be available at **http://localhost:5173**

### Build for Production

```bash
npm run build
```

Preview the production build:

```bash
npm run preview
```
